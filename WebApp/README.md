## Run Locally

Clone the project

```bash
  git clone https://github.com/shubhamkhuva/MashreqAssessment.git
```

Go to the project directory

```bash
  cd WebApp
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  npm run dev
```

Open http://localhost:3000 with your browser to see the result.
